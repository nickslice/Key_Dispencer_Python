/===============lang='ru'===============/
Директория 'dist' содержит файл приложения .exe
Для корректной работы программы в папке с файлом .exe должны находиться файлы с ключами с названиями batch_1.txt, batch_2.txt
При нажатии кнопки 'Get Key' создаётся файл с used ключами с отображением даты его использования
/===============lang='eng'===============/
The 'dist' directory contains the application .exe file
For correct work of the program the folder with the .exe file must contain files with keys named batch_1.txt, batch_2.txt
When you click the 'Get Key' button, a file with used keys is created with the display of the date of its use
